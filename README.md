## 🛠️ Software Engineering Blueprint (The IPO Engine)

This application completely avoids external UI libraries, utility frameworks, or tracking scripts to achieve optimal browser execution speeds by aligning directly with the **Universal Law of Software Engineering**:

1. **INPUT (Sensory Receptors):** The interface establishes a state of constant runtime vigilance using explicit event handling to capture immediate user stimuli—specifically watching keyboard interactions (`input` keystrokes), active window states, and lifecycle actions.
2. **PROCESS (The Business Logic Engine):** Employs isolated, strict contextual JavaScript routines to parse user text inputs, handle string validation, execute real-time array filtrations, and evaluate matching patterns.
3. **OUTPUT (The Target Mutation):** Directly modifies volatile tree map layouts in real-time, instantly rewriting data structures (`innerText`/`innerHTML`), toggling display states via utility class flags, and dynamically rendering text selection wrappers (`<mark>`) without page reloads.
